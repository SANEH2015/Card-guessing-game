# Card-guessing-game
## Run Locally

Clone the project
```bash
  git clone https://github.com/Amniei/Shopping-List.git
```
Go to the project directory
```bash
  cd Shopping-List
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run start
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
